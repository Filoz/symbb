# Bundles are used

- FOSUserBundle ( optional, but recommended )
- FOSRestBundle ( for future api )
- FOSJsRoutingBundle 
- KnpMenuBundle
- KnpPaginatorBundle
- SonataIntlBundle ( Date formating )
- LswMemcacheBundle (Memcache Manager)
